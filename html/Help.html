How to play:

Step 1: Download Novetus - here; Make sure that its the last version, it wont work if its not

Step 2: Download Novetus URI - When Novetus is at the "Play Novetus" screen, go to "other options" box, and select "Install URI" (You need to run Novetus as administrator)

Step 3: Download Radmin VPN - here

Step 4: Once Radmin VPN has set up, click on "join network". The network name is: NovetusProjectWinterServer. Password is: 123456

Step 5: After all that, you should be good to go - Select games tab and click join now button to enter them.

NOTE: on Radmin VPN, once youre on the network, you can see the SERVER-PC. If its offline then you cant join at the moment.

IMPORTANT: When you clikc play - A window should appear with your name and other configs. Press close to continue
